# pi-ynabro

Pi extension that registers [ynabro](https://www.npmjs.com/package/ynabro) tools for YNAB integration.

## Installation

```bash
pi install npm:pi-ynabro
```

## Available Tools

- `ynabro_get_pending_transactions`
- `ynabro_get_recent_transactions`
- `ynabro_approve_transaction`
- `ynabro_get_plan_info`

## Requirements

- `YNAB_TOKEN` environment variable must be set
